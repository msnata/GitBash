16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
grep - поиск
grep line file1.txt - поиск по слову line
grep -i line file1.txt - поиск с отключением регистрозависимости
grep -v 1 file1.txt - поиск всех строк, не содержащих 1
grep -с line file1.txt - посчет кол-ва строк со словом line
grep -ic line file1.txt - посчет кол-ва строк со словом line, с отключением регистрозависимости
grep -R line file1.txt - выводится список файлов, содержащий слово line
-n - Номера строк, в которых содержится искомое

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
